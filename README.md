# Calendario de Carreras

Aplicación web para registrar y gestionar tus carreras. Registra tus tiempos objetivo, tiempos reales, distancias y más.

## Características

- **Autenticación de usuarios**: Registro e inicio de sesión
- **Gestión de carreras**: Crear, editar y eliminar carreras
- **Vista de calendario**: Visualiza tus carreras en un calendario mensual
- **Vista de lista**: Lista todas tus carreras
- **Información completa**: 
  - Nombre de la carrera
  - Fecha
  - Tipo (calle, trail, montaña, postas, otro)
  - Distancia oficial y distancia real corrida
  - Tiempo objetivo y tiempo real
  - Notas adicionales

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Uso

1. Regístrate con tu email y nombre
2. Crea tu primera carrera
3. Completa todos los datos de la carrera
4. Visualiza tus carreras en el calendario o en la lista
5. Edita o elimina carreras según necesites

## Tecnologías

- React + TypeScript
- React Router
- date-fns para manejo de fechas
- LocalStorage para persistencia de datos
